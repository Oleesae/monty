# 0x19. C - Stacks, Queues - LIFO, FIFO

The objectives of this project include.
- The meaning of LIFO and FIFO
- Stack, Queues and when to use them
- Common implementations of stacks and queues
- Common uses of stacks and queues
- Proper ways to use global variables (extern)

## Monty Language.
 Monty 0.98 is a scripting language that is first compiled into Monty byte codes (just like python). It relies on a unique stack, with specific instructions to manipulate it.

- Therefore, the goal of this project is to create an interperter for Monty ByteCodes files.
